# Data Science & Analytics – Task 3: Marketing Funnel Analysis
**Future Interns Program (2026)**

## 📊 Project Overview
This project analyzes a multi-channel marketing funnel to trace the user journey from Website Visitors → Leads → Paying Customers. The goal is to identify critical drop-off points, calculate stage-by-stage conversion rates, and connect traffic data directly to revenue generation by evaluating Customer Acquisition Cost (CAC) and Return on Ad Spend (ROAS).

## 🎯 Answering the Core Business Questions
* **Where are users dropping off?** 85.5% of total traffic abandons the funnel at the very first stage (Visitor → Lead). 
* **Which channels bring high-quality leads?** Email Marketing is the highest quality, closing 40% of its generated leads into paying customers.
* **How can conversion rates be improved?** By reallocating budget away from broad Paid Social (which suffers from a $90 CAC) and into top-of-funnel lead capture strategies.
* **Which stages need optimization?** The main landing pages need A/B tested CTAs and "Lead Magnets" to fix the 85.5% initial bounce rate.

## 🛠️ Tools & Technologies Used
* **Python (Pandas):** Data structuring and KPI calculation (Conversion Rates, CAC, ROAS).
* **Plotly:** Interactive, presentation-ready dashboard visualizations.
* **Google Colab:** Environment for exploratory data analysis and executive reporting.
